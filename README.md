# Early Identity Assignment Digital Entities

This example demonstrates the concept of 'early identity definition' in digital systems. It shows how a unique identifier can be assigned to a digital entity (like an IoT device) at its earliest planning stage, before it is fully manufactured or configured. This pre-assigned ID then allows for consistent tracking and management throughout the entity's entire lifecycle.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
